西貢離島：隱藏於東岸的海上秘境

2026-06-08 1974 字
西貢 橋咀洲 吊鐘洲 甕缸群島 鹽田仔

在香港，如果想找一個既能遠離都市喧囂、又不會太過偏遠的目的地，西貢絕對是首選。這個位於新界東面的、社群型的小鎮，多年來一直是香港人「後花園」的概念存在——不是那種開發完善的旅遊區，而是保留了相當程度的慢節奏、空氣中是鹹鹹的海水味、岸邊停滿了灰色和白色的帆船。西貢本身不是島嶼，但它是通往多個美麗島嶼的門戶，這些島嶼面積雖小，特色卻十分鮮明，值得專程前往探索。

// 特色亮點

西貢商圈的島嶼最大的賣點是「未被過度商業化」。與長洲或南丫島相比，這裡的島嶼幾乎看不到手信店林立、也沒有大型餐廳連鎖進駐，維持著一種近似蠻荒的自然狀態。水質清澈是公開的秘密——東面海域不受維多利亞港水流影響，海底 能見度在夏天可達數米，船家常說「呢度游水可以去到對面山」。這些島嶼也是戶外活動愛好者的基地，每年夏季，碼頭旁邊聚集了不少自備呼吸管和麵包的人，等下一班渡輪前往自己的「秘密基地」。

// 推薦地點

【橋咀洲】

這應該是西貢最具人氣的島嶼，名字源於島上的一座民初石橋，現在只剩橋墩痕跡。島不大，環島步行一圈約一小時，重點是東岸的沙灘——不是那種開發完全的鋪裝沙灘，而是保留著原始面貌的碎石灘，水淺且平靜，很適合不太會游泳只是想泡泡水的人。島上沒有正規餐廳，只有碼頭邊一檔阿婆賣的山水豆腐花，HK$10一碗，豆香濃鬱，配上薑汁糖水，夏天喝十分消暑。需要注意的是，橋咀洲逢假日人流稍多，但相比港島南方的沙灘，仍屬「可以接受」的範圍。

【吊鐘洲】

如果橋咀洲是人氣選擇，吊鐘洲就是「內行人」的地盤。這個島的面積是西貢商圈中最大的之一，島上有多條行山路徑，最高點海拔約170米，天氣好時可以遠眺擔杆列島。吊鐘洲的沙灘集中在南端，水質公認是西貢群島中最佳的——沙灘面積不大，但海水質量穩定，常有浮潛客帶裝備下水。島上沒有任何商業設施，唯一的配套是一個、由嘉道理農場輔助經營的有機農場，提供簡單的茶水和本地種植的蔬菜。這個島更適合願意「自備糧草」的旅行者，否則要在船上解決午餐。

【甕缸群島】

嚴格來說，甕缸群島是由東甕、西甕和南甕三個小島組成的群島，名字來自島上形似甕缸的石堆。這裡是西貢最適合進階玩家的區域——船程較長（從西貢碼頭出發約40分鐘船程），所以人流最少，群島之間距離很近劃橡皮艇可以輕鬆來往。最著名的是「海底隧道」——兩塊巨巖之間的一條狹窄水道，漲潮時可讓小船透過，退潮時則變成一條可以行走的「海路」，是Instagram打卡的熱點。這個區域完全沒有商業設施，連飲水都必須自帶。

【鹽田仔】

這個小島的名字就是字面意思——昔日是產鹽的小島，如今已經荒廢，但在島的東面仍可看到昔日的鹽田痕跡。島很小，環島一週約20分鐘，適合「快速打卡」型的旅客，最大的亮點是島上的一座百年天后廟，據說是當年鹽民的信仰中心。碼頭邊有一艘固定泊岸的舊漁船，據船主說已經泊了三十多年，偶爾會賣自己釣的漁獲——能不能買到要看運氣。這個島的劣勢是船班較少，每天來回約八班，錯過要等下一班。

// 實用資訊

【交通】

從西貢市中心碼頭出發，所有島嶼均可乘坐街渡（俗稱「水上計程車」）抵達。街渡並非按固定班次執行，而是坐滿�人即開，所以旺季建議提前到達碼頭排隊。票價按目的地不同，橋咀洲單程約HK$35-40，吊鐘洲約HK$50，甕缸群島約HK$70，鹽田仔約HK$45。上船後直接在船上付款，不設找零，建議準備足夠零錢。也可包船前往，價格淡季約HK$800-1200每小時，旺季會加價兩成左右。

【費用】

橋咀洲門票：免費

吊鐘洲入場費：免費

鹽田仔參觀：免費

船上消費：山水豆腐花 HK$10 / 瓶裝水 HK$15/人造果汁 HK$25

租借浮潛裝備：HK$50/日（碼頭周邊有多檔）

【開放時間】

街渡服務：06:00-19:00（夏季旺季可能延長至20:00）

橋咀洲沙灘：24小時開放（但建議日落前離開）

吊鐘洲行山徑：日出至日落

// 旅遊小提示

西貢群島的旅行，準備工夫比實際行程更重要。首先，所有島嶼的餐飲選擇都非常有限，最多隻有碼頭邊的流動攤檔，強烈建議自備午飯和至少2升飲用水。其次，這些島嶼的移動訊號覆蓋不一，部分割槽域（如甕缸群島北面）完全沒有網路，自駕人士要先下載離線地圖。第三，最佳出行時間是平日的上午十點前——不僅人少，午後起風的話回程船隻可能會比較搖晃，容易暈船的人要有心理準備。最後，如果不是自駕船客，帶小朋友的家庭建議選擇橋咀洲——這裡的設施相對最完善，至少有基本的洗手間和救生員駐守。

西貢群島的另一個價值在於「彈性」。不像長洲有固定的tourist route，這裡更像是一個「自己去發現」的系統——每一次去都可能找到新的海灣、新的岩石形狀，或者新的漁民故事。這種「未知感」正是西貢在眾多離島中仍保持吸引力的原因。
