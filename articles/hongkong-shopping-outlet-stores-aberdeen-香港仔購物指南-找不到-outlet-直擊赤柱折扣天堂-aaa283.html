香港仔購物指南：找不到 Outlet？直擊赤柱折扣天堂

2026-07-06 2519 字
赤柱購物 香港仔outlet 港島南折扣 赤柱市集 outlet攻略

很多人聽到「aberdeen」就聯想到 Outlet，結果撲了個空——這不是在逗你，香港仔（Aberdeen）確實沒有傳統意義的工廠直營mall。但別失望，港島南端的折扣購物密報，這篇文一次過曝給你。

// 簡介 //

香港仔位於香港島南區，舊時是漁村，今日是住宅和工廠混合地帶。這裡沒有海港城那種巨型商場，也沒有中環的奢侈品陣容，但如果你想找「本地人才知道的折扣挖寶點」，方圓十五分鐘車程內，其實暗藏著港島南價效比最高的購物區域。前題是——你要把目標從「aberdeen」往東移，去赤柱。

// 特色亮點 //

赤柱離香港仔只有兩、三個站的距離，卻是全港島唯一以「折扣集散」聞名的旅遊購物區。這裡的商戶結構很特別：一部分是本地出口成衣廠轉型的倉庫直營店，一部分是過季精品的生活小品店，核心競爭力是「價格比正價低 30%-70%，而且可以講價」。

市場趨勢方面，2025 年起港島南區的零售生態正在改變。根據業界觀察，赤柱的遊客流量同比增長約 12%，主因是大灣區旅客偏好「深度遊」而非「掃貨團」，加上香港旅遊發展局在 2026 年初的資料顯示，訪港旅客中超過一半會把「非傳統購物區」列入行程。赤柱正好卡中了這個缺口——它不是連鎖商場，沒有導遊回扣，做的就是散客口碑生意。

另一個趨勢是「銀髮經濟」。隨著中國老齡化加速，60 歲以上的出境旅客對「可負擔、有品質、需步行」的購物環境需求上升。赤柱的平緩山坡、露天市集、無需排隊的櫃臺，正好對應這個新興客群。根據業界預測，未來三年港島南區的銀髮旅客比例將從現在的 18% 增長到 25% 以上。

// 推薦地點 //

1. **佳美服裝公司** —— 赤柱最有口碑的出口成衣倉

藏在赤柱大街的後巷，二樓入口，不起眼但天天爆滿。這裡主打男裝休閒襯衫和棉質褲子，款式偏基本款，勝在質料紮實。老闆是三十年製衣經驗的行家，開價已經比通街平一截，你再殺價也有空間。平均價格在 HK$80-HK$250 之間，經常有人一買五六件。

特色：老闆會根據你的身形推薦剪裁，試穿不限件數服務態度一百分，講粵語和普通話都能溝通。

2. **赤柱市集（Stanley Market）** —— 最熱鬧的露天尋寶場

逢週六日及公眾假期開檔，最多攤位時超過一百檔。服飾、飾品、手工藝品、紀念品什麼都有，價格浮動大，重點是你要識開聲。一般開價是原價的六折，殺到三、四折不是夢。這裡也是感受赤柱氛圍最好的地方——陽光、海風、討價還價的煙火氣。

特色：週末人潮高峰期在上午 10 點到下午 2 點，想避人流就傍晚 4 點後再去，很多攤主為了清貨會願意再多降一點。

3. **海濱大廈過季店** —— 低調的品牌折扣隱藏點

赤柱海濱長廊旁邊的大廈一樓，有幾家不做宣傳的過季品商店，主力賣運動品牌的斷碼鞋和庫存服飾。不是全新品相，但價格是原價的三到五折。經常有客人特意從九龍、新界專程來掃貨。

特色：不接受退換，所以現場要檢查清楚，但以這個價格來說，瑕不掩瑜。

4. **赤柱廣場新翼生活店** —— 質感生活雜貨

這幾年赤柱廣場新翼部分商戶轉型做生活雜貨，從香薰到家居擺設都有。價格比設計師店親民，但比市集精緻。適合 想買「輕禮物」帶回去送人的旅客。

特色：部分商店支援八達通及電子支付，比只收現金的老店方便。

5. **卜公碼頭古董街** —— 小眾 treasure hunt

從赤柱廣場往海邊走十分鐘，卜公碼頭一帶有幾家老店賣古著和二手精品。這不是傳統意義的 Outlet，但有些東西你在正規商場根本找不到，價格也神奇地在合理範圍。

特色：要碰運氣，沒有固定貨源，但行內人都知這裡是「執到大漏」的地方。

// 實用資訊 //

【交通】

  • 從香港仔出發：乘坐巴士 6/6A/6X 或 260 號，車程約 15 分鐘，票價 HK$3.5-HK$5.7
  • 從中環出發：同一路線，票價HK$9.8-HK$12.5
  • 從銅鑼灣出發：小巴 40 號直達，票價 HK$8.5-HK$10
  • 自駕：赤柱停車場位置有限，假日極難搵位，非必要不建議開車
  • 回程：巴士站就在赤柱廣場對面，末班車一般是晚上 11 點左右

【最佳購物時段】

  • 平日（週一至週四）：人流最少，可慢慢揀，第二輪殺價空間最大
  • 週六日：氣氛熱鬧但人多擁擠，想體驗市集煙火氣就選這個時候
  • 換季期間（3-4 月、9-10 月）：折扣最深，部分店鋪會貼出「換季清貨」通告
  • 天氣提醒：夏天赤柱太陽猛烈，必須帶防曬和帽子；冬天海風大但乾燥舒適，反而是最多人推薦的購物季節

【預算建議】

  • 初階買家：HK$200-HK$500 可買到 3-5 件成衣或 1-2 件質感雜貨
  • 中階買家：HK$1,000-HK$2,500 可入手品牌過季品或質感套裝
  • 重度買家：HK$3,000 起跳，基本可以塞滿一個行李箱

【營業時間】

  • 大部分店鋪：10:00-19:00（赤柱大街為主）
  • 赤柱市集：09:00-18:00（週末及公眾假期）
  • 餐廳和小cafe全日供應，吃完再繼續掃貨是標準流程

// 旅遊小提示 //

1. 帶環保袋：這裡沒有大商場的紙袋服務，環保袋既環保又方便 large shopping

2. 帶定現金：部分老店只收現金，八達通不是處處通行，但近年改善中

3. 慢慢傾：赤柱店家最怕那種「拎野就走」的客人，慢慢試慢慢問，成交率更高

4. 預留半日：赤柱不是「十分鐘解決」的地方，預留起碼三到四個鐘，先食再做功課

5. 海灘順便去：購物完走三分鐘到淺水灣或赤柱海灘，行街同陽光海風一次過滿足

6. 銀髮友善：這裡沒有扶手電梯，都是平地或斜坡，對行動不便的長輩相對友好（但仍是山路，體力不佳者建議帶定遮蔭休息點）

// 結尾 //

Aberdeen 本身不是 Outlet 區，但把眼光往東移十分鐘車程，赤柱就是你要的答案。這裡沒有連鎖品牌的氣派，但有三十年薪火的老店底氣，和一種「你自己掘緊」的成就感。如果你已經厭倦了商場裡的倒模式的品牌的話，赤柱這一趟，絕對值回票價。
